# MeltHand
流体シミュレーションのテストプロジェクト
手を溶かしてみた

# 目的
Unityで流体シミュレーションをすること
パッケージ仕様可

# 機能
- 手の形をした液体が溶ける
- 手のメッシュの透明度を調整可能
